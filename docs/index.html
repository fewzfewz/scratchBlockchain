Introduction to Nebula

A high-performance, modular blockchain built in Rust.

Nebula is designed for scalability and flexibility, featuring a multi-VM execution environment, validator-based consensus with instant finality, and built-in Layer 2 support.

High Performance

Parallel transaction execution and high-throughput mempool.

Secure

Ed25519 signatures and GRANDPA-style finality gadget.

Modular

Pluggable consensus, execution, and data availability layers.

Installation

Prerequisites: Rust (latest stable) and Docker.

# Clone the repository
git clone https://github.com/your-org/nebula.git
cd nebula

# Build release binary
cargo build --release

Quick Start

Deploy a local 3-node testnet using Docker Compose.

# Run deployment script
./scripts/deploy.sh

# Check status
curl http://localhost:9933/status

RPC API

Interact with the node via HTTP JSON-RPC.

GET /status

Returns current node status including block height and peer count.

POST /submit_tx

Submit a signed transaction to the mempool.
